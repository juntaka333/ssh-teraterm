# ssh-teraterm
Automatic acquisition of EC2 IP address

SSH　Client　：　TeraTerm

## Tutorial

1. Place AWS credentials on your own PC

2. edit ssh-client.ttl
```
[target-ec2-name] 
```
```
strconcat KEY_FILE '[/path/to/keyfile]'
```

3. connect server

click ssh-client.ttl

## Note

Instance name must be unique
